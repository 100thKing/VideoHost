# РГР Видеохостинг

## Запуск

### В папке **src**:
Windows
```
python -m venv venv
venv\Scripts\activate

pip install -r versions.txt

python manage.py runserver
```

Linux
```
python3 -m venv venv
source venv\bin\activate

pip3 install -r versions.txt

python3 manage.py runserver
```
